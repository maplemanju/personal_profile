MOBAゲームリーグ・オブ・レジェンドのカウンター情報共有所のサイトです。

* AJAX
* Riot API
* Transients API
* カスタムフィルド
* WPコメント

このサイトのコンテンツは、ユーザーとゲームのAPI、2つのソースから提供されています。
新しいバージョンの情報はゲームAPIから自動的にチェックされ、これは古いデータは新しいほうに切り替わって一時キャッシュ（WP Transient）に保存されます。
