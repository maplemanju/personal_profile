Faq Chat is a rule-based chat bot which means the messages are predefined and user the input comes from a group of selection. This is good to use on websites to help users with frequently asked questions.

Chat strings are pulled from json file as object. Its nodes have a pair of messages and choices, and the object gets updated every selection
made by the user.
