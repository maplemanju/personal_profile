チャット文字列は、オブジェクトとしてjsonファイルから取得されます。
そのノードにはメッセージと選択肢のペアがあり、オブジェクトはユーザーが選択するたびに更新されます。

* React
* Create React App
* JSX
* React Transition Group

ファクトボットは、ルールベースのチャットボットです。 メッセージは事前定義されており、ユーザー入力は選択のオプションから取得されます。 
これは、ビジターのよくある質問を答えるためにWebサイトで使用するのに適しています。
